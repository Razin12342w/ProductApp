# ProductApp

A React Native product listing application built using React Native CLI, TypeScript, Redux Toolkit, and DummyJSON API.

---

## Features

- Product Listing from API
- Product Detail Screen
- Search Functionality
- Infinite Scroll Pagination
- Redux Toolkit State Management
- React Navigation
- Functional Components & Hooks
- TypeScript Support

---

## Tech Stack

- React Native CLI
- TypeScript
- Redux Toolkit
- React Navigation
- DummyJSON API

---

## How to Run

Install dependencies:

```bash
npm install
```

Start Metro:

```bash
npx react-native start
```

Run Android app:

```bash
npx react-native run-android
```
